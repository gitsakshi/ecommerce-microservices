1️⃣ First Non-Repeating Character
Problem

Find first character that appears once.

Code (Java)
public char firstNonRepeating(String s) {
    Map<Character, Integer> map = new HashMap<>();

    for (char c : s.toCharArray()) {
        map.put(c, map.getOrDefault(c, 0) + 1);
    }

    for (char c : s.toCharArray()) {
        if (map.get(c) == 1) return c;
    }

    return '_'; // none
}

What interviewer checks

HashMap usage

clean logic

edge cases

2️⃣ Group Transactions by userId
Problem

Group list by userId.

Code
Map<Long, List<Transaction>> grouped =
    transactions.stream()
        .collect(Collectors.groupingBy(Transaction::getUserId));

Without streams (also good)
Map<Long, List<Transaction>> map = new HashMap<>();

for (Transaction t : transactions) {
    map.computeIfAbsent(t.getUserId(), k -> new ArrayList<>()).add(t);
}

What they check

Collections knowledge

data grouping logic
